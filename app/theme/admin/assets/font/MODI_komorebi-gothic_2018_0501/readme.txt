MODI工場／倒神神倒

梅フォントの「梅ゴシック」と「梅Hyゴシック」をお借りし、
オリジナルのかなを組み合わせアレンジした梅フォント派生フォントです。

梅フォント作者様に深く御礼申し上げます。

「デジタル」と「先生が黒板に書いた字」を混ぜて割ったようなイメージ。

等幅とプロポーショナルの２種です。
